 # Provenance Guard

 Provenance Guard is a backend service for classifying submitted creative content (text) to determine whether it was human-created or AI-generated. It provides multi-signal detection, confidence scoring with uncertainty, transparency labels, an appeals workflow, rate limiting, and structured audit logging.

 This repository contains the starter backend and deployment instructions.
